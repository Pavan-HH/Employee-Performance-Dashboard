# Employee-Performance-Dashboard

This Power BI Dashboard provides a comprehensive analysis of employee performance across different departments and time periods. It is designed to help HR teams, managers, and executives monitor key performance indicators (KPIs) and make data-driven decisions to improve productivity and engagement.

Key Insights:

Employee Overview: Displays total employees, active status, and department distribution.

Performance Metrics: Tracks performance scores, efficiency rates, and goal achievements.

Department Analysis: Compares performance across departments to identify top-performing teams.

Attendance & Punctuality: Visualizes attendance trends, leaves, and punctuality rates.

Trend Analysis: Monitors monthly and quarterly performance changes over time.

Tools & Technologies:

Power BI for dashboard development and visualization.

Excel / CSV Data Sources for input data.

DAX for calculated measures and KPIs.
